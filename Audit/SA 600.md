# SA 600: Using the Work of Another Auditor

## Objective
- To establish standards for principal auditor (Para 2)
  - Reporting on entity financial information (Para 2)
  - Using work of another auditor on components (Para 2)

## Scope
- Applies to: Principal auditor using other's work (Para 2)
- Does NOT apply to: Joint auditors (Para 3)
- Does NOT apply to: Predecessor auditor relationship (Para 3)
- ⚠️ Excludes immaterial components unless material in aggregate (Para 4)
- Effective from: April 1, 2002 (Para 26)

## Definitions
- 📌 Principal auditor (Para 6): "The auditor with responsibility for reporting on the financial information of an entity when that financial information includes the financial information of one or more components audited by another auditor."
- 📌 Other auditor (Para 7): "An auditor, other than the principal auditor, with responsibility for reporting on the financial information of a component which is included in the financial information audited by the principal auditor."
- 📌 Component (Para 8): "A division, branch, subsidiary, joint venture, associated enterprises or other entity whose financial information is included in the financial information audited by the principal auditor."

## Requirements
- Audit Impact and Acceptance
  - 🎯 The principal auditor should determine audit impact (Para 5)
    - Of using the other auditor's work (Para 5)
  - 🎯 The auditor should consider own participation (Para 9)
    - Is it sufficient to act as principal (Para 9)
  - 🎯 The principal auditor should consider professional competence (Para 11)
    - ⚠️ If other auditor is not ICAI member (Para 11)
- Principal Auditor Procedures
  - Right to visit and examine component books (Para 10)
    - ⚠️ Unless special circumstances make it essential (Para 10)
    - Normally entitled to rely on appointed auditor (Para 10)
    - 🔗 See SA 200 for reliance principles (Para 1)
  - 🎯 The principal auditor should perform procedures (Para 12)
    - To obtain sufficient appropriate audit evidence (Para 12)
    - That other auditor work is adequate (Para 12)
  - May discuss procedures or review summary (Para 13)
    - ⚠️ Not necessary if prior quality control evidence (Para 14)
  - 🎯 The principal auditor should consider significant findings (Para 15)
    - Of the other auditor (Para 15)
  - May decide supplemental tests are necessary (Para 16)
  - Above procedures assume added importance (Para 17)
    - If component auditor is unqualified foreign person (Para 17)
- Co-ordination Between Auditors
  - 🎯 There should be sufficient liaison between auditors (Para 19)
    - May issue written communications (Para 19)
  - 🎯 The other auditor should co-ordinate (Para 20)
    - Knowing context of work usage (Para 20)
  - 🎯 The other auditor should respond to questionnaires (Para 21)
    - When required by principal auditor timely (Para 21)
- Documentation Obligations
  - 🎯 The principal auditor should document working papers (Para 18)
    - ⚠️ Need not document reasons for limiting procedures (Para 18)
- Reporting and Responsibility
  - 🎯 The principal auditor should express qualified/disclaimer (Para 22)
    - ⚠️ If unable to perform sufficient additional procedures (Para 22)
  - 🎯 The principal auditor should consider report modification (Para 23)
    - ⚠️ If other auditor modifies their report (Para 23)
  - Principal is not responsible for other's work (Para 24)
    - ⚠️ Except if suspicion aroused regarding reliability (Para 24)
  - 🎯 Report should state division of responsibility clearly (Para 25)
    - Indicating extent of components audited by others (Para 25)

## Application Guidance
- No explicit application paragraphs exist (Para 1-26)
  - SA 600 follows pre-clarity formatting structure (Para 1)

## Key Nuances and Exceptions
- ⚠️ Para 4: Immaterial components excluded
  - If immaterial in isolation: Procedures do not apply
  - If material in aggregate: Procedures should be considered
- ⚠️ Para 10: Right of reliance
  - Condition met: Normally entitled to rely
  - Condition not met: Must visit if special circumstances
- ⚠️ Para 14: Discussing procedures
  - If prior quality control known: Application not necessary
- ⚠️ Para 24: Responsibility
  - Default state: Principal not responsible
  - If suspicion aroused: Principal becomes responsible

## ICAI Listed Examples
- Para 9 — Considerations for principal auditor role
  - 📋 Materiality of portion audited by principal auditor
  - 📋 Degree of knowledge regarding component business
  - 📋 Risk of material misstatements in components
  - 📋 Performance of additional procedures
- Para 12 — Items to advise other auditor
  - 📋 Use to be made of work and report
  - 📋 Areas requiring special consideration
  - 📋 Procedures for identifying inter-component transactions
  - 📋 Time-table for completion of audit
  - 📋 Significant accounting, auditing and reporting requirements
- Para 18 — Items for documentation
  - 📋 Components audited by other auditors
  - 📋 Significance to entity financial information
  - 📋 Names of the other auditors
  - 📋 Conclusions that individual components are not material
  - 📋 Procedures performed and conclusions reached
  - 📋 How modified remarks were dealt with
- Para 20 — Co-ordination by other auditor
  - 📋 Bringing significant findings to immediate attention
  - 📋 Adhering to the time-table for audit
  - 📋 Ensuring compliance with statutory requirements
- Para 25 — Indicating division of responsibility
  - 📋 Number of divisions, branches, subsidiaries, or components

## Logical Flow Chains
- Limitations on Scope (Para 22)
  - Work of other auditor cannot be used
    - requires: Performance of additional procedures
      - if unable to perform: Express qualified or disclaimer opinion
- Component Modification Impact (Para 23)
  - Other auditor issues modified report
    - leads to: Consider nature and significance
      - if significant to entity: Modify principal report
